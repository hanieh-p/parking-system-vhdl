# Automatic Parking Gate Controller

## Description
This project implements an automatic parking gate controller using VHDL.

The system detects vehicles using sensors, controls the gate opening and closing process, and displays the available parking spaces on a seven-segment display.

## Features
- Vehicle detection using sensors
- Automatic gate control
- Seven-segment display output
- Modular VHDL design
- Testbench for simulation

## Tools
- Language: VHDL
- Simulator: Active-HDL

## Project Structure
- counter.vhd
- sensor.vhd
- gate control.vhd
- seven segment.vhd
- top level.vhd
- test bench.vhd
